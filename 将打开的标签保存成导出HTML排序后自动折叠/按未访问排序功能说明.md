# 按未访问排序功能说明

## 功能描述
在生成的HTML文件中添加了"按未访问排序"按钮,该功能会:
1. 按照规则分组显示标签
2. 只显示未访问过的标签(没有访问记录的标签)
3. 保留规则分组的结构和颜色

## 实现方式
1. 在视图控制按钮中添加"按未访问排序"按钮
2. 创建新的视图容器 `byRulesUnvisited`
3. 在页面加载时,根据localStorage中的访问记录过滤标签
4. 只显示没有访问记录的标签,按规则分组展示

## 使用方法
1. 保存标签后打开生成的HTML文件
2. 点击"按未访问排序"按钮
3. 查看按规则分组的未访问标签列表

## 注意事项
- 未访问的标签是指在localStorage中没有访问记录的标签
- 如果清除了访问历史,所有标签都会显示为未访问
- 该功能依赖于浏览器的localStorage,不同浏览器的数据不共享

## 修改内容

### 1. 添加视图按钮
在视图控制区域添加新按钮:
```html
<button class="view-button" data-view="byRulesUnvisited">按未访问排序</button>
```

### 2. 添加视图容器
在views区域添加新容器:
```html
<div class="tabs-container" id="byRulesUnvisited">
  ${ruleGroupsUnvisitedHTML}
</div>
```

### 3. 生成未访问标签的规则分组HTML
添加新函数 `generateRuleBasedGroupsUnvisited()` 来生成只包含未访问标签的规则分组视图
